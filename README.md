# Clinic Duty Scheduler

Система автоматической генерации графиков дежурств сотрудников клиники на Django.

Проект предназначен для создания, управления и автоматического распределения дежурств с учетом:

- типа сотрудника (основной / совместитель);
- нормы рабочих часов;
- пожеланий сотрудников по доступности дат;
- типов смен;
- автоматического распределения нагрузки.
